Pour tester le proto dans des conditions optimales, il faut veiller à différencier la version PC et la version Android
Une fois qu'un PC a installé le jeu et qu'au moins un téléphone (4 max) ont l'apk :

- Lancer le jeu sur PC
- Cliquer sur Host : un code apparaît
- Lancer le jeu sur téléphone
- Taper le code sur le téléphone en faisant attention à la casse
- Cliquer sur Join (une connexion Internet stable est requise)

Un cube représentant le joueur apparaît sur l'écran d'ordinateur
Deux features sont implémentées : un mini-jeu et la création du crush

CRÉATION DU CRUSH :
- Sur le PC, cliquer sur "Start (crush)" : le Crush est visible à l'écran
- Pour le modifier, il faut cliquer sur le bouton blanc du téléphone "Start Crush Creation" (au milieu)
- Deux flèches apparaissent sur le téléphone
- Cliquer sur les flèches permet de faire défiler des attributs prédéfinis
- Chaque joueur peut modifier seulement une partie
- Pour fermer cette fenêtre, cliquer sur le bouton en bas à droite sur le PC (/!\ il vous sera impossible d'y retourner)

MINI-JEU :
- Sur le PC, cliquer sur "Start (game)"
- Des obstacles surgissent sur l'écran
- Cliquez sur le bouton "Jump" sur votre téléphone pour faire sauter votre personnage
- Le mini-jeu redémarre après quelques temps
